# Health+

## About the Project
Cuidar+ is a web application designed to support elderly well-being by providing accessible and user-friendly tools to improve autonomy and quality of life.

## 🚀 Features
- Medication reminders
- Hydration tracking
- Cognitive activities
- Accessible interface
- Voice support (in development)

## 🛠 Technologies
- TypeScript
- HTML
- CSS
- Tailwind

## Purpose
To create an accessible solution that helps elderly individuals manage their daily routines, promoting independence and well-being through technology.

## 🌍 Impact
This project was inspired by real visits to nursing homes, focusing on solving real-world problems faced by elderly people.
